# 人民币汇率时序分析与预测项目
## 一、项目简介
本项目以**1994-2025年人民币汇率中间价**为研究对象，开展时序数据分析与多模型预测。
依次完成数据清洗、时序可视化、平稳性检验、差分处理，使用 **ARIMA、指数平滑、随机森林** 三类模型进行短期预测，并通过 MAE、MSE、RMSE 量化对比模型效果。

## 二、数据来源
数据文件：`data/rmb_exchange_rate.xlsx`
数据来源：**国家外汇管理局**https://www.safe.gov.cn/safe/2020/1218/17833.html
时间范围：1994年 — 2025年（日度人民币兑多币种中间价）
包含币种：美元、欧元、日元、港元、英镑等26种主流货币。

## 三、项目文件说明
### 1. 根目录文件
- `requirements.txt`：Python运行依赖库清单
- `AI_Record.md`：项目开发、代码调试、问题解决的AI交互记录

### 2. code 代码文件夹
1. `01_data_process.py`：数据读取、日期格式转换、排序、缺失值填充、数据清洗与导出
2. `02_analysis.py`：时序趋势可视化、ADF平稳性检验、一阶差分分析
3. `03_model_arima.py`：ARIMA时序模型训练、预测与指标评估
4. `04_model_smooth.py`：霍尔特指数平滑模型训练、预测与指标评估
5. `05_model_rf.py`：随机森林回归模型、特征工程、多模型汇总评估
6. `06_summary_metrics.py`：批量提取三类模型评价指标，生成汇总表格与可视化对比图

### 3. 其他文件夹
- `data/`：存放原始Excel数据、清洗后CSV数据
- `result/`：自动保存所有可视化图表、模型评估结果表格
